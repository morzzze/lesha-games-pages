Пауза

Игра остановлена. Нажми Продолжить, чтобы вернуться в матч.

Для iPhone удобнее играть в горизонтальном режиме. Поверни телефон набок.
Управление: WASD / стрелки — движение, Space — действие, P — пауза. Импостер атакует рядом, мирный делает рывок.
