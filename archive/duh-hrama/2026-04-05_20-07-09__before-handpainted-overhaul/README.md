# Дух Храма

Отдельный HTML-проект для браузерной мини-игры про бой с боссом в храме.

## Что внутри
- `index.html` — текущая рабочая версия игры в одном файле

## Текущее состояние
- чистый HTML/CSS/JS без внешних зависимостей
- открывается как обычный файл в браузере
- адаптировано под iPhone / Android / desktop
- есть HP игрока, HP босса, комбо, победа / поражение и фаза ярости
- визуальный стиль обновлён в сторону более мрачной и мистической сказочной атмосферы
- добавлен усиленный cinematic pass: слои тумана, глубина фона, более выразительная маска духа и атмосферный overlay

## Важно
- исходная версия была прислана в чат как HTML с Three.js
- текущая версия пересобрана в single-file canvas-игру без внешних библиотек
- если придёт более свежая версия, текущий файл можно заменить, а прошлую версию убрать в `projects/archive/html/duh-hrama/`

## Как запускать
Самый простой способ — открыть `index.html` в браузере.
