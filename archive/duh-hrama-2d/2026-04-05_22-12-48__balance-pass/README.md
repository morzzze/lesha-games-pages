# Дух Храма 2D

Отдельный HTML-проект для архивной 2D/canvas-ветки игры «Дух Храма».

## Что внутри
- `index.html` — текущая рабочая версия игры в одном файле

## Новое в этой итерации
- после победы над первой формой запускается автоматическая сцена: дух взлетает и храм рушится
- затем без ручной кнопки начинается бой с true boss / giant deadly creature
- во второй форме босс кидается книгами, стреляет молниями и стрелами
- по арене падают щиты-подборы
- стрелы можно отражать обратно в босса, если есть щит

## Текущее состояние
- чистый HTML/CSS/JS без внешних зависимостей
- открывается как обычный файл в браузере
- адаптировано под iPhone / Android / desktop
- есть HP игрока, HP босса, комбо, победа / поражение и фаза ярости
- визуальный стиль обновлён в сторону более мрачной и мистической сказочной атмосферы
- добавлен усиленный cinematic pass: слои тумана, глубина фона, более выразительная маска духа и атмосферный overlay
- сделан hand-painted overhaul: более детализированная сцена, колонны, арка храма, алтарь, персонаж, снаряды и HUD

## Важно
- эта ветка создана из архивной 2D/canvas single-file версии
- она отдельна от Three.js-ветки `duh-hrama`
- если придёт более свежая версия именно 2D-ветки, текущий файл можно заменить, а прошлую версию убирать в `projects/archive/html/duh-hrama-2d/`

## Как запускать
Самый простой способ — открыть `index.html` в браузере.
