# Дух Храма 2D — balance pass

Отдельный HTML-проект для 2D/canvas-ветки игры «Дух Храма».

## Что внутри
- `index.html` — текущая рабочая версия игры в одном файле

## Новое в этой итерации
- уменьшено число маленьких книжных монстров в волнах
- снижена скорость и урон книжных монстров
- замедлена и ослаблена гигантская форма
- уменьшено количество книг и молний в залпах
- сохранены 4 стрелы, но снижены их скорость и урон
- игроку дан больший запас HP и больше щитов
- щиты чаще выпадают и сильнее помогают выиграть бой

## Важно
- эта ветка создана из архивной 2D/canvas single-file версии
- она отдельна от Three.js-ветки `duh-hrama`
- если придёт более свежая версия именно 2D-ветки, текущий файл можно заменить, а прошлую версию убирать в `projects/archive/html/duh-hrama-2d/`

## Как запускать
Самый простой способ — открыть `index.html` в браузере.
