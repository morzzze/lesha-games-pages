# Дух Храма 2D — journey preboss pass

Отдельный HTML-проект для 2D/canvas-ветки игры «Дух Храма».

## Что внутри
- `index.html` — текущая рабочая версия игры в одном файле

## Новое в этой итерации
- перед храмом добавлен новый пролог-путь
- герой приплывает на одинокий остров на лодке
- на берегу встречают книжные монстры, которые теперь активно охотятся на героя и кидаются книгами
- дальше идёт путь к храму через ловушки и прыжки
- затем спуск в шахты с падающими камнями и вагонетками
- перед входом в храм добавлен новый мини-босс: `Хранитель шахт`
- после победы над ним открывается уже знакомый вход в храм и основной бой
- мобильное управление расширено кнопкой `ПРЫЖОК`
- исправлен критичный баг островной засады: береговые книжные монстры теперь корректно спавнятся, двигаются, преследуют героя и атакуют книгами, так что береговой бой больше не выглядит пассивным
- при получении урона герой больше не исчезает полностью: вместо этого даётся короткий полупрозрачный flash во время invulnerability
- сделан fairness-balance pass: базовое HP игрока увеличено, урон книжных монстров и боссов снижен, а при входе в полосы препятствий, в шахты и перед крупными боями здоровье полностью восстанавливается и выдаются щиты
- в journey-сегментах добавлен мягкий checkpoint-respawn вместо жёсткого поражения, чтобы прохождение не ломалось из-за одной ошибки
- для телефонов добавлен anti-freeze pass: облегчён hit-fx при получении урона и включён fail-safe recovery в основном frame loop
- сделан mobile-first HUD/UI pass: верхний HUD стал компактнее, приоритеты экрана собраны вокруг HP / прогресса / ресурса боя, а кнопки управления перестроены под более чистые thumb-zones
- герой стал визуально меньше, чтобы не съедать полезную игровую область на телефоне
- сделан visual identity pass: боссы на разных этапах теперь различаются сильнее, а береговые монстры и храмовые волны получили разные силуэты и визуальные архетипы
- после шахт добавлен новый акт: выход в джунгли, болотная переправа с лианами, деревяшками и крокодилами, затем бой с болотным духом на `1200 HP`
- за победу над болотным духом герой получает усиленное заклинание на `400` урона перед храмовым контуром
- сегмент `Тропа к храму` стал понятнее и мягче: убран автоскролл без движения, уменьшен урон ловушек, сужены опасные зоны и подсказка про прыжок стала явнее
- ямы на тропе перестроены в безопасные трещины/обвалы, чтобы участок не застревал на непроходимых прыжках

## Текущее состояние
- чистый HTML/CSS/JS без внешних зависимостей
- открывается как обычный файл в браузере
- адаптировано под iPhone / Android / desktop
- теперь есть цельный путь: остров → книжные монстры → ловушки → шахты → мини-босс → храм → основной босс
- spell balance pass и храмовый бой сохранены

## Важно
- эта ветка создана из архивной 2D/canvas single-file версии
- она отдельна от Three.js-ветки `duh-hrama`
- для следующих итераций брать за source-of-truth именно `projects/html/duh-hrama-2d/index.html`

## Как запускать
Самый простой способ — открыть `index.html` в браузере.

## Проверка
1. Нажать `ПРИСТАТЬ К ОСТРОВУ`
2. Дождаться высадки и отбиться от книжных монстров
3. Пройти ловушки, используя движение и прыжок
4. Спуститься в шахты и пережить вагонетки / падающие камни
5. Победить `Хранителя шахт`
6. Коснуться печати и убедиться, что храмовый бой и spell всё ещё работают
