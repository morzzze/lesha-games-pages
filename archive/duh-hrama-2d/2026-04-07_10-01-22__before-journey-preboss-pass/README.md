# Дух Храма 2D — spell balance pass

Отдельный HTML-проект для архивной 2D/canvas-ветки игры «Дух Храма».

## Что внутри
- `index.html` — текущая рабочая версия игры в одном файле

## Новое в этой итерации
- spell теперь заряжается не с `10`, а с `12` комбо
- spell по клавише `F` и мобильной кнопке `ОБРЯД` всё ещё работает
- burst-урон по боссу снижен с `400` до `320`
- радиус spell по книжным волнам уменьшен, чтобы заклинание не сносило слишком много врагов сразу
- индикатор готовности в HUD обновлён до `0/12 → ГОТОВО`
- мобильная кнопка обновлена до `12X`

## Текущее состояние
- чистый HTML/CSS/JS без внешних зависимостей
- открывается как обычный файл в браузере
- адаптировано под iPhone / Android / desktop
- есть HP игрока, HP босса, комбо, победа / поражение, true boss и сбалансированная spell-атака
- визуальный стиль остаётся мрачным и мистическим
- single-file архитектура сохранена для быстрых итераций и публикации

## Важно
- эта ветка создана из архивной 2D/canvas single-file версии
- она отдельна от Three.js-ветки `duh-hrama`
- для следующих итераций брать за source-of-truth именно `projects/html/duh-hrama-2d/index.html`

## Как запускать
Самый простой способ — открыть `index.html` в браузере.

## Проверка
1. Набрать `11` комбо и убедиться, что spell ещё не готов
2. Набрать `12` комбо и увидеть `ГОТОВО · F`
3. Нажать `F` на desktop или кнопку `ОБРЯД` на телефоне
4. Проверить, что босс получает `320` урона, а по книжным волнам spell всё ещё чистит ближайшую группу, но не слишком широко
