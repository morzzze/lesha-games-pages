# Дух Храма 2D — journey preboss pass

Отдельный HTML-проект для 2D/canvas-ветки игры «Дух Храма».

## Что внутри
- `index.html` — текущая рабочая версия игры в одном файле

## Новое в этой итерации
- перед храмом добавлен новый пролог-путь
- герой приплывает на одинокий остров на лодке
- на берегу встречают книжные монстры
- дальше идёт путь к храму через ловушки и прыжки
- затем спуск в шахты с падающими камнями и вагонетками
- перед входом в храм добавлен новый мини-босс: `Хранитель шахт`
- после победы над ним открывается уже знакомый вход в храм и основной бой
- мобильное управление расширено кнопкой `ПРЫЖОК`
- исправлен критичный баг островной засады: береговые книжные монстры теперь корректно спавнятся, двигаются и бой больше не застревает

## Текущее состояние
- чистый HTML/CSS/JS без внешних зависимостей
- открывается как обычный файл в браузере
- адаптировано под iPhone / Android / desktop
- теперь есть цельный путь: остров → книжные монстры → ловушки → шахты → мини-босс → храм → основной босс
- spell balance pass и храмовый бой сохранены

## Важно
- эта ветка создана из архивной 2D/canvas single-file версии
- она отдельна от Three.js-ветки `duh-hrama`
- для следующих итераций брать за source-of-truth именно `projects/html/duh-hrama-2d/index.html`

## Как запускать
Самый простой способ — открыть `index.html` в браузере.

## Проверка
1. Нажать `ПРИСТАТЬ К ОСТРОВУ`
2. Дождаться высадки и отбиться от книжных монстров
3. Пройти ловушки, используя движение и прыжок
4. Спуститься в шахты и пережить вагонетки / падающие камни
5. Победить `Хранителя шахт`
6. Коснуться печати и убедиться, что храмовый бой и spell всё ещё работают
